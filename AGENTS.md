# AGENTS.md Рђћ QuenchWorks website

Instructions for AI agents working in this repo. Human contributors: see `README.md`.

## What this is

The public site for **QuenchWorks** Рђћ a free, independent catalog of hardened, 0-CVE
container images and signed Helm charts (a vendor-neutral Bitnami-catalog replacement).
It is a **marketing + catalog + docs** site, register **`brand`** (design carries the
identity). Strategy lives in **`PRODUCT.md`**; the visual system in **`DESIGN.md`** Рђћ read
both before any UI work.

Stack: **Astro** (static, no SSR adapter) ┬и **Tailwind CSS v4** (tokens in
`src/styles/global.css` `@theme`, no `tailwind.config`) ┬и self-hosted Inter + JetBrains
Mono ┬и Pagefind search ┬и i18n **en / ar / es** (Arabic is RTL). Package manager: **pnpm**.

## Commands

| Command | What it does |
|---|---|
| `pnpm dev` | Local preview (runs `predev` Рєњ `sync` first). **Validate every UI change here before committing.** |
| `pnpm build` | `astro build` + `seal-redirects` + Pagefind index. Must pass before commit. |
| `pnpm check` (`npx astro check`) | Typecheck. **Must be 0 errors / 0 warnings / 0 hints** before commit. |
| `pnpm sync` | Regenerate `src/data/{images,charts}.json` + runtime dockerfiles + OG cards from `../images/catalog.lock.yaml`. |
| `pnpm scan` (`node scripts/scan-images.mjs`) | Trivy-scan every published image Рєњ `src/data/security.json` (also runs nightly in CI). |

## Layout

- `src/pages/` Рђћ routes. English at root, `/ar` and `/es` locale trees. Includes `api/` (the JSON API + shields badges), `images/`, `charts/`, `docs/`.
- `src/components/` Рђћ Astro components (`Hero`, `Catalog`, `CveStatus`, `CatalogList`, `ImageDetail`, `ChartDetail`, `RuntimeDetail`, `Header`, `Footer`, Рђд).
- `src/layouts/` Рђћ `Base.astro` (head, i18n `dir`/hreflang, JSON-LD, skip link), `Docs.astro`.
- `src/data/` Рђћ see "Generated data" below. `artifacthub.ts` is the data-access layer.
- `src/content/` Рђћ `changelog` (+ `changelog-ar`, `changelog-es`) and `docs` (MDX).
- `src/i18n/ui.ts` Рђћ all UI strings, keyed, in **en / ar / es**.
- `src/styles/global.css` Рђћ Tailwind v4 `@theme` tokens (the palette lives here).

## Generated data Рђћ do NOT hand-edit

- `src/data/images.json`, `src/data/charts.json` Рђћ generated by `pnpm sync` from `../images/catalog.lock.yaml`. To change catalog content, edit the source in the **images** repo (`catalog.yaml` + `gen-catalog.py`) and re-run sync Рђћ never edit these JSON files directly.
- `src/data/security.json` Рђћ generated by the nightly Trivy scan (`.github/workflows/security-scan.yml` / `pnpm scan`). Real per-image grades/severities; **never hardcode CVE numbers** anywhere.
- `src/data/dockerfiles/`, `public/og/` Рђћ generated by `pnpm sync`.
- `src/data/roadmap.json` Рђћ hand-maintained (priority `next` / `planned` / `exploring`).

## Conventions

- **Design system is normative.** Monochrome dark (ink/graphite/line/ash/paper). No new colors or fonts. Color appears **only** to encode CVE severity (emerald/amber/red) and is **always** backed by a text grade + `aria-label`. Flat Рђћ separation is a 1px `line` border + tonal step, never a shadow. See `DESIGN.md` for the bans (no gradient text, no decorative glass, no cream bg, no per-section uppercase eyebrow, no side-stripe borders).
- **i18n.** Every user-facing string goes through `t()` in `src/i18n/ui.ts` and must be translated in **all three** locales (en/ar/es). Never ship an untranslated string; Arabic must read correctly RTL.
- **Accessibility.** WCAG 2.1 AA. Keyboard nav + visible `:focus-visible` rings; honor `prefers-reduced-motion` and `prefers-contrast`; tabular figures for any numeric data.

## Hard rules

- **Validate before commit.** Do not commit website/UI changes until they've been checked in `pnpm dev`. Wait for the go-ahead.
- **Commit trailer.** Every commit ends with exactly:
  `Co-Authored-By: QuenchWorks <info@quench-works.com>` Рђћ **never** a Claude / AI-assistant trailer.
- `astro check` (0/0/0) and `pnpm build` must pass before any commit.
- **Revert incidental churn.** Don't ship `package.json` / `pnpm-lock.yaml` dependency bumps as part of a content or UI change; `git checkout` them unless the bump is the point of the change.
- `main` is the deploy branch; the site auto-deploys on push via the host's git integration. Push only when asked.
